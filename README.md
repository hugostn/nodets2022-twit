# Tiago Gomes Sant'Ana

## Installation

Requeriments:

- [node.js](https://nodejs.org)
- [yarn](https://yarnpkg.com)

First time install:

```sh
# install dependencies
yarn
# build
yarn build
# start server
yarn start
```

## Development
```sh
# install dependencies
yarn
# start server in dev watch mode
yarn dev
```
